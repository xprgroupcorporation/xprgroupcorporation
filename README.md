## 👋 Who are we?

XPR is an independent startup from Thailand building software, games, and developer tools. (for now.)  

We're a small team with ambitious ideas, focused on creating products that are useful, scalable, and built to last. Growth matters,  
but so does trust. We believe good products should earn support, not rely on aggressive monetization or unnecessary complexity. 

This GitHub is home to **XPR Studio™**, where we build, experiment, and share the technology behind our projects.  

> **Made with intent. Built to last. Built for the world.**
